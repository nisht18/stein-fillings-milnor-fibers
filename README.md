# Stein Fillings and Milnor Fibers of 4-Manifolds

An academic repository documenting independent research, literature synthesis, and mathematical derivations in low-dimensional and contact topology. This work builds upon the Plamenevskaya-Starkston framework to study constraints on Stein fillings of contact 4-manifolds using singularity theory and Lefschetz fibrations.

This repository serves as a reproducible archive of my mathematical preparation for graduate-level research.

---

## 🔬 Core Research Focus

This project explores the intersection of geometric topology and singularity theory through three main pillars:
1. **Algebraic & Topological Constraints:** Investigating the limitations placed on Stein fillings of 4-manifolds using invariants arising from the Plamenevskaya-Starkston framework.
2. **Singularity Theory Invariants:** Analyzing the topology of Milnor fibers to derive explicit topological bounds on allowed Stein fillings.
3. **Fibration Derivations:** Calculating deformation and filling structures combinatorially via planar open books and Lefschetz fibrations.
